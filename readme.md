# Gridless

A [less CSS](lesscss.org) 12-column grid system.
Based on [1140 CSS Grid](http://cssgrid.net/) and [Inuit CSS](http://csswizardry.com/inuitcss).

## Basic Usage

```html
...
<div class="container">
  <div class="row">
    <div class="col three">
      Three columns
    </div>
    <div class="col six">
      Six columns
    </div>
    <div class="col three">
      Three columns
    </div>
  </div>
</div>
...
```

## Configuration

Configuration options can be set in `grid-config.less`
